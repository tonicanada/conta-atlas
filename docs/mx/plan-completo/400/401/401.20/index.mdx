## Descripción formal
La cuenta 401.20 Ingresos por servicios profesionales nacionales partes relacionadas se utiliza para registrar los ingresos generados por prestaciones de servicios profesionales que se realizan dentro del territorio nacional y entre entidades relacionadas, es decir, entre empresas afiliadas o filiales.

## Explicación práctica
Esta cuenta se aplica cuando una empresa presta servicios profesionales a otra empresa con la cual existe un vínculo de relación, como pueden ser sociedades matriz-hijo, empresas asociadas o cualquier otro tipo de entidad relacionada. Es importante distinguir este ingreso del que se registra en 401.10, que corresponde a servicios prestados a clientes no relacionados.

## Relación con otros nodos
- **Padre**: 401 Ingresos.
- **Hermanos**: 401.01 a 401.12 (otros tipos de ingresos).
- **Nodos inferiores**: No se especifican en el contexto proporcionado.

## Errores comunes
Un error común es confundir esta cuenta con otras subcuentas del grupo 401, como 401.10 (servicios a extranjeros) o 401.07 (exentos), lo que puede llevar a un mal registro de ingresos y afectar la precisión financiera de las transacciones. Es crucial identificar correctamente el tipo de cliente para aplicar la cuenta adecuada.
