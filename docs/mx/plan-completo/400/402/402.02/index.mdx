## Descripción formal
La cuenta 402.02, "Devoluciones, descuentos o bonificaciones sobre ventas y/o servicios al 0%", pertenece a la categoría de Devoluciones, descuentos o bonificaciones sobre ingresos y se clasifica como una subcuenta (leaf) del grupo 402.

## Explicación práctica
Esta cuenta registra las devoluciones, descuentos o bonificaciones que se aplican a las ventas y/o servicios cuando la tasa impositiva es cero. Es importante distinguir esta cuenta de otras subcuentas del mismo nivel, ya que refleja transacciones específicas donde no se aplica ningún impuesto.

## Relación con otros nodos
- **Padre**: 402 — Devoluciones, descuentos o bonificaciones sobre ingresos.
- **Hermanos**: 
  - 402.01 — Devoluciones, descuentos o bonificaciones sobre ventas y/o servicios a la tasa general.
  - 402.03 — Devoluciones, descuentos o bonificaciones sobre ventas y/o servicios exentos.
  - 402.04 — Devoluciones, descuentos o bonificaciones de otros ingresos.

## Ejemplos
Esta cuenta se utiliza para registrar situaciones específicas donde las transacciones no imparten impuesto, como:
- Descuento en la facturación de un servicio de mantenimiento que es exento.
- Devolución parcial de una venta donde el producto no genera impuestos.
- Bonificación otorgada a un cliente por cumplir ciertos requisitos sin implicar impuestos.

## Errores comunes
1. **Confusión con otras subcuentas**: Es común que se confunda esta cuenta con 402.01 (devoluciones, descuentos o bonificaciones sobre ventas y/o servicios a la tasa general) o 402.03 (devoluciones, descuentos o bonificaciones sobre ventas y/o servicios exentos). Es crucial identificar el contexto de cada transacción para aplicar la cuenta correcta.
2. **Inclusión de impuestos**: Algunas veces, los usuarios pueden incluir en esta cuenta devoluciones que sí generan impuestos, lo cual es incorrecto y puede llevar a errores fiscales.
3. **Omitir registros**: Olvidarse de registrar estos descuentos o bonificaciones puede resultar en un balance incompleto y potenciales problemas contables.
