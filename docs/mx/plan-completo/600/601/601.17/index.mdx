## Descripción formal
La cuenta 601.17 — Servicio médico se clasifica como una subcuenta bajo el rubro de gastos generales, específicamente destinada a registrar los costos asociados con servicios médicos proporcionados o pagados al personal del establecimiento.

## Explicación práctica
Esta cuenta se utiliza para documentar y controlar los gastos relacionados con la atención médica que se brinda a los empleados. Esto incluye consultas, medicamentos, exámenes médicos, entre otros servicios de salud necesarios para el bienestar del personal laboral.

## Relación con otros nodos
- **Padre**: 601 — Gastos generales
- **Hermanos**: Cuentas relacionadas con gastos de nómina y administrativos, como sueldos y salarios (601.01), compensaciones (601.02), tiempos extras (601.03) y otros.
- **Subcuentas**: No existen subcuentas directas para 601.17, aunque puede que se utilicen códigos más específicos en el sistema contable interno del establecimiento.

## Ejemplos
- Se registra el costo de un chequeo médico anual realizado a todos los empleados.
- Se documentan los gastos por medicamentos prescritos para un empleado.
- Se incluyen los costos asociados con la atención médica en caso de accidente laboral.

## Errores comunes
1. **Incorporación de gastos no relacionados**: Es importante que solo se registren aquellos gastos directamente relacionados con servicios médicos, evitando confusiones con otros rubros como seguros o vacaciones.
2. **Omisión de costos**: No olvidar registrar todos los servicios médicos proporcionados a los empleados para mantener un registro completo y preciso.
3. **Confusión con otros cuentas**: Asegurarse de no confundir esta cuenta con otras relacionadas con la salud, como 601.08 (prima dominical) o 602 (gastos médicos directos).
