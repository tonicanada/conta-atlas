## Descripción formal
La cuenta 601.54 Limpieza se clasifica como una subcuenta bajo el rubro de gastos generales, específicamente destinada a registrar los costos asociados con la limpieza y mantenimiento del ambiente laboral.

## Explicación práctica
Esta cuenta es fundamental para mantener un registro preciso de los recursos destinados a la conservación del entorno de trabajo. Incluye servicios como el lavado de pisos, desinfección, recolección de basura, limpieza de equipamiento y áreas comunes.

## Relación con otros nodos
- **Padre**: 601 — Gastos generales
- **Hermanos**: Cuentas relacionadas con gastos operativos del personal (sueldos, compensaciones) y otras subcategorías de gastos generales.
- **Subcuentas**: No se especifican subcuentas directas para 601.54.

## Ejemplos
- Se registra el pago a un servicio externo de limpieza diaria del edificio principal.
- Se anota la compra de productos químicos y suministros necesarios para realizar las tareas de mantenimiento.
- Se incluye el costo de contratación temporal de personal para una campaña de desinfección especial.

## Errores comunes
Es común que se confunda esta cuenta con otras relacionadas a gastos operativos. Es importante distinguir entre la limpieza y otros rubros como mantenimiento estructural o servicios profesionales. Además, asegurarse de que todos los costos asociados con el mantenimiento del ambiente laboral sean correctamente asignados a 601.54 para una correcta clasificación contable.
