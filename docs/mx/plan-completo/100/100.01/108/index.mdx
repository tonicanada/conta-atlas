## Descripción formal
La cuenta 108 "Estimación de cuentas incobrables" es un agrupador que sirve para estimar las posibles pérdidas asociadas a cuentas por cobrar que podrían no ser recuperadas. Esta cuenta permite una clasificación más detallada entre cuentas incobrables nacionales e internacionales, así como entre partes relacionadas y no relacionadas.

## Explicación práctica
Esta cuenta es crucial para la gestión financiera de una empresa, ya que ayuda a anticipar posibles pérdidas en el futuro. Al estimar estas cuentas incobrables, se pueden ajustar los estados financieros de manera más precisa, reflejando situaciones reales y mejorando la solvencia del balance.

  - 108.02 — Estimación de cuentas incobrables extranjero: Para cuentas incobrables fuera del país.
  - 108.03 — Estimación de cuentas incobrables nacional parte relacionada: Para partes relacionadas dentro del país.
  - 108.04 — Estimación de cuentas incobrables extranjero parte relacionada: Para partes relacionadas fuera del país.

- Este agrupador no incluye ejemplos operativos detallados, ya que se trata de un agrupador.

## Errores comunes
1. **No asignar correctamente**: No distribuir las estimaciones entre los diferentes subgrupos (nacional vs extranjero y parte relacionada vs no relacionada).
2. **Subestimar o sobreestimar**: No ajustar adecuadamente la estimación de cuentas incobrables, lo que puede llevar a una representación inexacta del estado financiero.
3. **Olvidar revisar regularmente**: No realizar revisiones periódicas para actualizar las estimaciones, lo cual podría resultar en información obsoleta y desactualizada.
