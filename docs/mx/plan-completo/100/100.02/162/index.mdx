## Descripción formal
La cuenta 162 "Embarcaciones" es un agrupador del nivel 3 dentro de la categoría de activos a largo plazo, específicamente destinada para registrar y controlar los bienes muebles náuticos utilizados por la empresa. Esta cuenta sirve como contenedor para subcuentas que detallan diferentes tipos o unidades de embarcaciones.

## Explicación práctica
Esta cuenta se utiliza para agrupar todas las embarcaciones propiedad de la empresa, independientemente del uso específico que se les dé (por ejemplo, transporte de mercancías, turismo, investigación, etc.). Es importante mantener esta cuenta actualizada para un mejor control y análisis financiero de los activos náuticos.

## Relación con otros nodos
- **Padre**: 100.02 — Activo a largo plazo: Esta cuenta se encuentra en el nivel 3 bajo la categoría general de activos a largo plazo.
- **Hermanos**: Las cuentas hermanas incluyen terrenos, edificios, maquinaria y equipo, entre otros, que representan diferentes tipos de activos fijos. La cuenta 162 se destaca como un tipo específico dentro del grupo de activos fijos.
- **Hijos**: La subcuenta 162.01 — Embarcaciones: Esta cuenta detalla específicamente las embarcaciones, permitiendo una subdivisión más precisa del patrimonio náutico.

## Ejemplos
Aunque la cuenta es de tipo agrupador (group), se pueden mencionar ejemplos generales:
- Inversión en un yate para uso turístico.
- Adquisición de un barco pesquero para operaciones comerciales.
- Compra de una lancha de investigación marina.

## Errores comunes
1. **Incorporación incorrecta**: No incluir embarcaciones en esta cuenta, sino en otras subcategorías que no corresponden.
2. **Omitir actualizaciones**: Olvidar actualizar la información cuando se realicen cambios en el patrimonio de las embarcaciones.
3. **Confusión con otros activos**: Confundir embarcaciones con otros tipos de activos fijos, como maquinaria o edificios náuticos.
