## Descripción formal
La cuenta 502.01 Compras nacionales es una subcuenta del grupo de cuentas 502 — Compras, destinada a registrar las transacciones relacionadas con la adquisición de bienes y servicios dentro del país.

## Explicación práctica
Esta cuenta se utiliza para documentar todas las compras realizadas en el territorio nacional, incluyendo materias primas, insumos, equipos y servicios que se requieren para el funcionamiento normal de la empresa. Es fundamental para mantener un control preciso sobre los gastos internacionales, permitiendo una mejor gestión financiera y cumplimiento con las obligaciones fiscales.

## Relación con otros nodos
- **Padre**: 502 — Compras: Esta cuenta se ubica directamente bajo el grupo de cuentas 502.
- **Hermanos**: 
  - 502.02 — Compras nacionales parte relacionada: Cuenta que registra las compras internacionales que tienen un vínculo directo con la empresa, pero no se incluyen en esta cuenta.
  - 502.03 — Compras de Importación: Se utiliza para registrar todas las compras realizadas fuera del país.
  - 502.04 — Compras de Importación partes relacionadas: Cuenta que registra las compras internacionales relacionadas con la empresa, pero específicamente de proveedores extranjeros.

## Errores comunes
1. **Incluir compras internacionales en esta cuenta**: Es importante distinguir entre compras nacionales y de importación para evitar confusiones.
2. **Omitir registros de compras nacionales**: Es crucial mantener un registro preciso de todas las transacciones locales para cumplir con la contabilidad y la fiscalidad.
3. **Confundir con cuentas similares**: Asegúrate de no confundir esta cuenta con otras subcuentas del grupo 502, como las que se refieren a compras internacionales.

Esta descripción proporciona una guía clara sobre cómo utilizar la cuenta 502.01 en el plan de cuentas SAT mexicano, asegurando su correcto uso y evitando errores comunes.
