## Descripción formal
La cuenta 502.04 Compras de Importación partes relacionadas se utiliza para registrar las transacciones de compras realizadas a empresas relacionadas en el marco del comercio internacional.

## Explicación práctica
Esta cuenta es relevante para comprender y controlar los costos asociados con la importación de bienes desde proveedores que tienen una relación de parentesco, control o influencia. Es crucial para evaluar la eficiencia operativa y potencialmente las prácticas comerciales entre empresas relacionadas.

## Relación con otros nodos
- **Padre**: 502 — Compras
- **Hermanos**: 
  - 502.01 — Compras nacionales
  - 502.02 — Compras nacionales parte relacionada
  - 502.03 — Compras de Importación

Esta cuenta se sitúa bajo el nodo 502.03 Compras de Importación, junto con otras subcuentas que registran compras internacionales.

## Ejemplos
- Ejemplo 1: Una empresa importa componentes electrónicos de una filial en otro país para su uso en la producción.
- Ejemplo 2: Un proveedor local es adquirido por el grupo empresarial, y posteriormente se realiza una compra de bienes importados desde esta nueva filial.
- Ejemplo 3: Se contrata a un proveedor extranjero que está bajo el control de una empresa matriz del mismo grupo.

## Errores comunes
1. **Incluir compras nacionales en esta cuenta**: Es importante distinguir entre las compras realizadas dentro del país y aquellas que involucran importaciones.
2. **Omitir la identificación de partes relacionadas**: No se debe confundir con otras subcuentas de importación que no involucren a empresas relacionadas.
3. **No registrar transacciones internacionales**: Es fundamental incluir todas las compras realizadas fuera del país, incluso si son de empresas afiliadas o controladas por el grupo empresarial.
