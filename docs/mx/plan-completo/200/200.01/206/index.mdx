## Descripción formal
La cuenta 206, "Anticipo de cliente", es un agrupador que se encuentra en el nivel 3 del plan de cuentas SAT bajo la categoría de pasivo a corto plazo. Esta cuenta sirve para agrupar y controlar los anticipos recibidos de clientes.

## Explicación práctica
Esta cuenta se utiliza para registrar y gestionar los anticipos o depósitos efectuados por clientes antes del cumplimiento total de las obligaciones contractuales. Es importante distinguir esta cuenta de otras relacionadas con pasivos a corto plazo, ya que su propósito es específicamente para los anticipos recibidos.

## Ejemplos
Aunque se trata de un agrupador, aquí se ilustra cómo se pueden utilizar sus subcuentas:

- **206.01 Anticipo de cliente nacional**: Se registra el anticipo recibido de un cliente nacional que ha realizado una venta anticipada.
- **206.03 Anticipo de cliente nacional parte relacionada**: Se utiliza para registrar anticipos dados a clientes que son partes relacionadas, como filiales o socios comerciales.

## Errores comunes
1. Confundir el anticipo de cliente con otros pasivos a corto plazo.
2. No registrar correctamente los anticipos recibidos, lo que puede llevar a una mala gestión financiera.
3. Olvidar transferir los anticipos a cuentas específicas según la naturaleza del cliente o la relación comercial.
