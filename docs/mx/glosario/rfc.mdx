# RFC

## Definición breve
El RFC es un número de registro fiscal que identifica única y exclusivamente a las personas físicas o morales en México para efectos fiscales.

## Explicación práctica
El RFC, que se pronuncia "Raf" (abreviatura de Registro Federal de Contribuyentes), es un código alfanumérico de 13 dígitos que sirve como identificador único para las personas físicas y morales en el sistema tributario mexicano. Este número es requerido para realizar operaciones fiscales, contratar servicios públicos, y cumplir con obligaciones tributarias.

## Ejemplo
Un ejemplo de un RFC válido sería: AAA010101HHH, donde "AAA" representa la entidad federativa (por ejemplo, Aguascalientes), "010101" es el número de identificación del contribuyente y "HHH" son los dígitos verificadores.

## Se usa en… / Relacionado con…
- Registros Federales
- Impuestos federales (IVA, ISR)
- Operaciones financieras y comerciales
- Servicios públicos
