# Buzón Tributario

## Definición breve
El Buzón Tributario es un sistema electrónico del Servicio de Administración Tributaria (SAT) que permite a los contribuyentes enviar y recibir información relacionada con sus obligaciones fiscales.

## Explicación práctica
Este sistema facilita el intercambio de datos entre las empresas y el SAT, permitiendo la transmisión de documentos como declaraciones de ingresos, certificaciones, solicitudes de información y otros documentos necesarios para cumplir con los requisitos tributarios. Los contribuyentes pueden utilizar este servicio para enviar sus documentos en formato digital, lo que agiliza el proceso administrativo y reduce errores.

## Ejemplo
Un ejemplo práctico sería cuando una empresa necesita enviar su declaración de ingresos al SAT. En lugar de imprimir y enviar la documentación por correo postal, puede hacerlo a través del Buzón Tributario, donde se suben los archivos digitales necesarios para cumplir con el plazo fiscal.

## Se usa en… / Relacionado con…
- Proceso de envío de declaraciones fiscales
- Gestión de certificaciones tributarias
- Solicitud de información fiscal
