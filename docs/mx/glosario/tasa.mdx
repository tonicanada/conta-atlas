# Tasa

## Definición breve
La tasa se refiere al porcentaje o proporción que se aplica para calcular impuestos, tasas de interés, descuentos, entre otros.

## Explicación práctica
En el contexto de los impuestos en México, la tasa es un valor expresado en porcentaje que se aplica a una base imponible para determinar el monto del tributo. Esta tasa puede variar según el tipo de impuesto y las leyes fiscales vigentes.

## Ejemplo
Si una persona tiene un ingreso bruto anual de 500,000 pesos y la tasa de ISR (Impuesto Sobre la Renta) es del 35%, entonces el monto a pagar en impuestos sería de 175,000 pesos.
