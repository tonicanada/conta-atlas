# Estado de resultados

## Definición breve
El estado de resultados, también conocido como estado de ganancias o pérdidas, es un informe financiero que muestra las operaciones económicas realizadas por una empresa durante un período determinado.

## Explicación práctica
Este documento refleja la capacidad de una empresa para generar ingresos y controlar sus gastos. Se divide generalmente en dos secciones: ingresos y gastos, con el objetivo de determinar si la empresa ha tenido ganancias o pérdidas durante el período analizado.

## Ejemplo (si aplica)
Un ejemplo simplificado podría ser:
```
Ingresos:
- Venta de productos: $500,000
- Servicios profesionales: $200,000

Gastos:
- Costo de bienes vendidos: $300,000
- Salarios y sueldos: $150,000
- Alquiler: $50,000
- Gastos publicitarios: $20,000

Resultado neto: $180,000
```
