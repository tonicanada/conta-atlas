# Base gravable

## Definición breve
La base gravable es la cantidad sobre la cual se calcula el impuesto a pagar. Esencialmente, representa el valor del bien o servicio que está sujeto al pago de un tributo.

## Explicación práctica
En términos prácticos, la base gravable puede variar según el tipo de impuesto y las leyes fiscales aplicables en México. Por ejemplo, para el Impuesto Sobre Nómina (ISN), la base gravable es el salario neto del trabajador; mientras que para el Impuesto al Valor Agregado (IVA), la base gravable puede incluir o no los descuentos y deducciones según las reglas específicas de cada caso.

## Ejemplo
Supongamos un negocio que vende productos con IVA. Si el precio total de venta es de $1,000 y se aplica una tasa del 16% de IVA, la base gravable sería $1,000 (el monto total de la venta).
