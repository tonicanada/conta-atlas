# CUCA

## Definición breve
CUCA es un acrónimo que se utiliza en contextos relacionados con impuestos en México, aunque no es directamente asociado con el ISR. En este contexto interno, se refiere a la **Cuenta de Utilidades Contables**.

## Explicación práctica
La Cuenta de Utilidades Contables (CUCA) es un elemento contable que registra las ganancias y pérdidas netas de una empresa durante un período fiscal. Esta cuenta es fundamental para el cálculo del Impuesto Sobre la Renta (ISR), ya que sirve como base para determinar los ingresos gravables.

## Ejemplo
Por ejemplo, si una empresa reporta utilidades por $500,000 y pérdidas por $100,000 durante un año fiscal, su CUCA mostrará un saldo neto de utilidades de $400,000. Este monto se utilizaría para determinar el ISR a pagar.

## Se usa en… / Relacionado con…
- Cálculo del ISR
- Informes financieros anuales
- Declaración de Impuestos Sobre la Renta (DIR)
